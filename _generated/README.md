# `_generated/` — do not hand-edit

Everything in this directory is machine-generated by [`../scripts/sync.mjs`](../scripts/sync.mjs) from sources the Lettr team already maintains:

| Output | Built from |
|-|-|
| `sdk/<lang>/<topic>.md` | `../lettr-docs/quickstart/<lang>/*.mdx` |
| `api/index.md` | `../lettr-docs/openapi.json` (condensed) |
| `api/openapi.json` | `../lettr-docs/openapi.json` (copied verbatim) |

Each file carries an `AUTO-GENERATED … do not edit` header.

**To change this content, edit the upstream docs/SDK, then run `npm run sync` and commit the result.** Hand-edits here are overwritten on the next sync and are flagged by `npm run sync:check` in CI.

These files are committed on purpose: end users install the skill without access to `../lettr-docs` or the SDK repos, so the generated output must ship in the package.
